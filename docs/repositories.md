## 结构理解

目前可以把这些仓库理解为三层：

- 官网层：`openilink.com`
- SDK 层：不同运行时的接入包
- 生态 / 中枢层：`openilink-hub`、`openilink-tg`

## 使用建议

- 如果你要开始接入，优先从 [SDK 总览](/sdk/) 进入。
- 如果你要了解组织全貌，优先从当前页面浏览所有仓库。
- 如果你要参与文档或品牌层建设，直接查看 `openilink.com`。
