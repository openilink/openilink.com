# 仓库

下面是 `openilink` 组织当前公开的仓库列表。这个页面用于把官网、Hub、SDK 与生态项目放到同一个索引面里，避免入口分散。当前同步到的公开仓库总数为 10 个，其中 SDK 仓库为 6 个。

## 结构理解

目前可以把这些仓库理解为四层：

- 官网层：`openilink.com`
- 中枢层：`openilink-hub`
- 渠道 / 生态层：`openilink-tg`、`openclaw-channel-openilink`
- SDK 层：6 个不同运行时的接入包

## 使用建议

- 如果你想直接进入平台表面，先打开 [Hub](https://hub.openilink.com)。
- 如果你要开始接入，优先从 [SDK 总览](/sdk/) 进入。
- 如果你要了解组织全貌，优先从当前页面浏览所有仓库。
- 如果你要参与文档或品牌层建设，直接查看 `openilink.com`。
