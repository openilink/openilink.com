# Java SDK

Java SDK 面向 JVM 生态，适合希望在 Java 或 Spring 服务中接入 OpeniLink 的团队。

## 仓库

[openilink/openilink-sdk-java](https://github.com/openilink/openilink-sdk-java)

## 定位

Java SDK 负责把 OpeniLink 的接入能力带到 JVM 运行时中，并与其他语言 SDK 共同围绕统一的消息链路模型演进。

## 适用场景

- Java / Spring 服务端系统
- 偏向稳定工程体系和长期维护的项目
- 需要和其他多语言服务共同接入 OpeniLink 的团队

## 当前建议

Java SDK 已经公开，现阶段建议先直接查看仓库进展；官网会随着实现逐步补充更完整的接入说明与示例。
