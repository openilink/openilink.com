# Java SDK

Java SDK 面向 JVM 生态，是 OpenILink SDK 矩阵中新加入的一条运行时线路。

## 仓库

[openilink/openilink-sdk-java](https://github.com/openilink/openilink-sdk-java)

## 定位

Java SDK 负责把 OpenILink 的接入能力带到 Java 服务端环境中，并与其他语言 SDK 共同围绕统一的消息链路模型演进。

## 适用场景

- Java / Spring 服务端系统
- 偏向稳定工程体系和长期维护的项目
- 需要和其他多语言服务共同接入 OpenILink 的团队

## 当前建议

由于 Java SDK 目前是最新纳入组织矩阵的一条线路，建议先直接查看仓库进展；官网会随着实现逐步补充更完整的接入说明。
