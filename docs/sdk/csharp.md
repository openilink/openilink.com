# C# SDK

C# SDK 面向 .NET 生态，适合希望在企业级后端或 Windows / 跨平台服务环境中集成 OpeniLink 的团队。

## 仓库

[openilink/openilink-sdk-csharp](https://github.com/openilink/openilink-sdk-csharp)

## 定位

C# SDK 负责把 OpeniLink 的核心接入模型带到 .NET 运行时中，重点仍然是保持与其他语言 SDK 的方法和流程一致。

## 适用场景

- .NET / ASP.NET 服务
- 企业内部系统集成
- 偏向静态类型和成熟工程工具链的团队

## 当前建议

目前建议优先关注仓库本身的实现状态，官网先提供统一入口与定位说明，后续再补充更细的 API 和示例文档。
