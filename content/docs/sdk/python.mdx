Python SDK 面向脚本化能力、服务端集成和更偏自动化的应用场景。

## 仓库

[openilink/openilink-sdk-python](https://github.com/openilink/openilink-sdk-python)

## 定位

Python SDK 是 OpeniLink 多语言接入矩阵的一部分，目标是在 Python 生态下提供与其他运行时尽量一致的能力模型。

## 适用场景

- Python 服务端与任务系统
- 自动化工具或集成脚本
- 需要快速把消息链路接入到既有 Python 项目的团队

## 当前建议

如果你的主栈是 Python，先从仓库本身查看当前实现与示例会更直接，后续官网会随着 SDK 成熟度继续补充独立文档。
