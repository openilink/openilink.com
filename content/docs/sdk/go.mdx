Go SDK 面向希望在静态类型和原生并发模型中完成接入的团队。

## 仓库

[openilink/openilink-sdk-go](https://github.com/openilink/openilink-sdk-go)

## 定位

Go SDK 是官方多语言 SDK 矩阵的一部分，目标是与其他运行时保持一致的接入动作：

- 登录
- 长轮询收消息
- 发消息
- 配置获取
- 上下文管理

## 适用场景

- Go 服务端系统
- 偏向轻依赖和可部署性的后端项目
- 希望与 Node.js / PHP SDK 共享一致接入流程的团队

## 当前建议

如果你的项目已经在 Go 生态里，直接进入仓库查看当前实现与示例进度会更高效。
