本指南将带你从零开始，在本地部署 OpeniLink Hub 并发送第一条消息。

## 环境要求

在开始之前，请确保你的机器上已安装：

- [Docker](https://docs.docker.com/get-docker/) (20.10+)
- [Docker Compose](https://docs.docker.com/compose/install/) (v2+)

## 第一步：克隆仓库

```bash
git clone https://github.com/openilink/openilink-hub.git
cd openilink-hub
```

## 第二步：启动服务

```bash
docker compose up -d
```

这会启动三个服务：

| 服务 | 端口 | 说明 |
| --- | --- | --- |
| PostgreSQL 17 | 15432 | 数据持久化存储 |
| MinIO 对象存储 | 19000 (API) / 19001 (控制台) | 文件和媒体存储 |
| OpeniLink Hub | 9800 | Hub 主服务 |

<Callout type="info">
首次启动时，Hub 会自动执行数据库迁移，创建所有必要的表结构。无需手动运行迁移命令。
</Callout>

等待所有容器启动完成后，可以通过以下命令确认状态：

```bash
docker compose ps
```

## 第三步：访问 Hub 控制台

在浏览器中打开 [http://localhost:9800](http://localhost:9800)，你将看到 OpeniLink Hub 的登录页面。

## 第四步：注册账号

OpeniLink Hub 支持多种注册方式：

- **密码注册**：使用用户名和密码注册
- **Passkey 生物识别**：使用指纹、面部识别或硬件密钥
- **GitHub OAuth**：使用 GitHub 账号登录（需提前配置）
- **LinuxDo OAuth**：使用 LinuxDo 账号登录（需提前配置）

本地开发环境下，建议直接使用密码注册。

## 第五步：扫码绑定微信 Bot

登录后，按照以下步骤绑定你的微信 Bot：

1. 点击控制台中的 **"绑定 Bot"** 按钮
2. 页面将显示一个二维码
3. 用微信扫描该二维码
4. 在微信客户端确认登录
5. Bot 自动上线，系统自动创建默认 Channel

<Callout type="warn">
绑定微信 Bot 需要使用真实的微信账号。请确保该微信号未在其他 iLink 服务上同时登录。
</Callout>

## 第六步：创建 Channel 并获取 API Key

绑定 Bot 后，系统会自动创建一个默认 Channel。你也可以手动创建更多 Channel：

1. 进入 Bot 详情页
2. 点击 **"新建 Channel"**
3. 配置 Channel 名称和消息过滤规则
4. 创建完成后，复制该 Channel 的 **API Key**

API Key 是你通过 SDK 或 API 接入该 Channel 的凭证。每个 Channel 拥有独立的 API Key。

## 第七步：用 SDK 接收消息

以 Node.js 为例，安装 SDK 并编写接收消息的代码：

```bash
npm install openilink-sdk-node
```

```ts
import { Client, extractText } from "openilink-sdk-node";

// 使用 Channel API Key 初始化客户端
const client = new Client("<你的 API Key>");

// 扫码登录
const result = await client.loginWithQr({
  on_qrcode: (url) => console.log("请扫描二维码:", url),
});

if (!result.connected) {
  throw new Error(result.message);
}

console.log("Bot 已上线，开始监听消息...");

// 监听消息
await client.monitor(async (message) => {
  const text = extractText(message);
  if (!text) return;

  console.log(`收到消息: ${text}`);

  // 回复消息
  await client.sendText(
    String(message.from_user_id),
    `收到: ${text}`,
    String(message.context_token)
  );
});
```

运行代码后，向你的微信 Bot 发送一条消息，你将看到 Bot 自动回复。

## 环境变量说明

以下是 OpeniLink Hub 支持的全部环境变量：

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `LISTEN` | `:9800` | HTTP 监听地址 |
| `DATABASE_URL` | `postgres://openilink:openilink@localhost:15432/openilink?sslmode=disable` | PostgreSQL 连接字符串 |
| `RP_ORIGIN` | `http://localhost:9800` | WebAuthn RP 源地址 |
| `RP_ID` | `localhost` | WebAuthn RP ID |
| `SECRET` | `change-me-in-production` | 服务端密钥，用于签名和加密 |
| `STORAGE_ENDPOINT` | — | MinIO/S3 端点地址 |
| `STORAGE_ACCESS_KEY` | — | 存储访问密钥 |
| `STORAGE_SECRET_KEY` | — | 存储密钥 |
| `STORAGE_BUCKET` | `openilink` | 存储桶名称 |

<Callout type="error">
在生产环境中，务必修改 `SECRET` 为随机的强密码字符串，并正确配置 `RP_ORIGIN` 和 `RP_ID` 为你的实际域名。
</Callout>
